# 株式投資ラジオ 台本生成ルール

## 番組概要
- **番組名**: 「株式投資ラジオ — 90日で見つける自分の戦い方」
- **尺**: 約15分（日本語で約4,500〜5,000文字）
- **形式**: 一人語りポッドキャスト風（講師がリスナーに語りかける）
- **リスナー**: 有持雄陽さん（投資初心者、積立投資を少しやっている）
- **配信頻度**: 毎日（90日間のカリキュラム制）

## リスナープロフィール
- 外壁塗装会社のマーケ・開発担当
- 積立投資を少しやっているレベルの初心者
- AIを業務で高度に活用している（Claude, LINE Bot, GitHub Actions等）
- 建設・塗装業界の知識が豊富
- 学びたいこと：基礎から、自分に合った投資スタイルを見つけたい

## トーン・スタイル
- **信頼感のある講師**が隣で話しているような親しみやすさ
- 敬語ベースだがフレンドリー
- 「今日のテーマは〜です」で始める
- 難しい金融用語は必ず例え話で説明
- 「有持さんの業務に例えると〜」のように、塗装業やAI活用と絡めた例え
- 具体的な数字やシミュレーションを多用（抽象論ではなく実感を持たせる）
- 「これ、テストに出ます」的な強調で重要ポイントを印象付ける

## 台本構成

### 1. オープニング（約1分）
```
おはようございます！株式投資ラジオ、Day{N}の放送です。
今日のテーマは「{レッスンタイトル}」。
{テーマの導入・なぜこの話が重要か}
```

### 2. メインレッスン（約11分）
各トピックについて：
1. **概念の説明**: わかりやすく
2. **具体例・数字**: リアルな例で実感を持たせる
3. **身近な例え**: 塗装業・ビジネス・日常生活に例える
4. **よくある誤解**: 初心者がハマりがちなポイント

### 3. 今日のまとめ（約2分）
```
今日のキーメッセージ: {key_message}
覚えておいてほしいポイントを3つに絞ると...
```

### 4. エンディング（約1分）
```
明日のDay{N+1}では「{次のレッスンタイトル}」をお届けします。
{次回の簡単な予告}
それでは、今日も良い投資家ライフを！
```

## 講義の原則
1. **正確性**: 金融知識は正確に。推奨銘柄の具体名は出さない
2. **中立性**: 特定の投資スタイルを押し付けない。メリット・デメリット両方伝える
3. **段階性**: カリキュラムの進行度に合わせた難易度で話す
4. **実用性**: 「聴いたらすぐ行動できる」レベルの具体性
5. **継続性**: 前回の復習を入れつつ、新しい内容に進む

## 出力フォーマット
- 台本: `Day{N}_{YYYY-MM-DD}_台本.md`
- 読み上げ用: `Day{N}_{YYYY-MM-DD}_読み上げ.txt`
  - マークダウン記法を除去
  - 英語の固有名詞はカタカナのみ
  - 数字はそのまま使用OK
  - 自然な話し言葉で句読点を入れる
